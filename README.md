# رحلة إلى أجمل ذكرى ✨

تجربة ويب تفاعلية سينمائية — هدية عيد ميلاد. مبنية بـ React 19 + Vite + Tailwind + Framer Motion + GSAP + Firebase.

## حالة التسليم — الجزء 1 من 2

هذا الجزء يتضمن المشروع كاملًا بهيكله ونظامه (routing، الحالة، الصوت، الإنجازات، Firebase)، مع اكتمال كامل لأربعة أقسام:

- ✅ المشهد السينمائي الافتتاحي (سماء ليلية، نجوم، قمر، نص متدرج)
- ✅ صندوق الهدية الأول (اهتزاز → فتح → جسيمات احتفالية)
- ✅ مشهد المائدة (حوار لوسي، زر "لا" الهارب، صحن تمن وقيمة، شمعة)
- ✅ خريطة الرحلة التفاعلية (تفتح المراحل بالتتابع)
- ✅ شخصية لوسي الأصلية بالكامل (SVG مرسومة يدويًا، رمش تلقائي، تعبيرات، تلويح)
- ✅ نظام الجسيمات (نجوم / يراعات / بتلات) على Canvas بأداء 60fps
- ✅ نظام الصوت (مؤثرات طبيعية فقط، بدون موسيقى) مع زر كتم
- ✅ نظام الإنجازات + التخزين المحلي + تكامل Firebase لنجمة الأمنيات

الأقسام التالية (كاميرا الذكريات، نجمة الأمنيات، الرسائل، لعبة لوسي، صندوق الصور، الهدية الأخيرة) هي حاليًا صفحات مؤقتة بسيطة، وستصلكِ كاملة الميزات في الرسالة التالية — دون حذف أو اختصار أي ميزة مطلوبة.

## التشغيل محليًا

```bash
npm install
npm run dev
```

## البناء للنشر

```bash
npm run build
```

سينتج مجلد `dist/` جاهز للنشر مباشرة على GitHub Pages (تم ضبط `base: './'` في `vite.config.js` ليعمل من أي مسار فرعي).

### النشر على GitHub Pages

```bash
npm run deploy
```

(يتطلب تثبيت `gh-pages`، وهو مُدرج ضمن `devDependencies`.)

## إعداد Firebase (لنجمة الأمنيات)

1. أنشئي مشروعًا على [Firebase Console](https://console.firebase.google.com).
2. فعّلي Firestore Database.
3. انسخي `.env.example` إلى `.env` واملئي القيم من إعدادات مشروعك.
4. إن تُرك `.env` فارغًا، ستعمل نجمة الأمنيات تلقائيًا بالتخزين المحلي (localStorage) دون أي عطل.

## هيكل المشروع

```
src/
  components/   عناصر واجهة قابلة لإعادة الاستخدام (لوسي، الجسيمات، الأزرار...)
  pages/        كل مشهد/مرحلة من الرحلة كصفحة مستقلة
  context/      إدارة الحالة العامة (التقدّم، الصوت)
  firebase/      تهيئة Firebase وخدمة الأمنيات
  data/          تعريف المراحل والإنجازات
  styles/        التنسيق العام وRTL
```

## الصوتيات المطلوبة

ضعي ملفات mp3 خفيفة جدًا (نسيم، عصافير، فتح صندوق، خطوات، ورق، رنين، نقرة) داخل:
`src/assets/audio/` بالأسماء: `breeze.mp3`, `birds.mp3`, `box-open.mp3`, `footsteps.mp3`, `paper.mp3`, `chime.mp3`, `click.mp3`
